CoffeeTime update: Family Access

This version adds a Family Access section. In synced mode, the first Google user becomes Owner automatically. The Owner can add other approved emails in CoffeeTime > Family.

Recommended use:
1. Upload all files to GitHub.
2. Wait for Cloudflare Pages to deploy.
3. Sign in with your own Google account first.
4. Open Family and add your girlfriend's email.
5. After confirming both users can access the app, add stricter Firestore Rules from the Family page.
